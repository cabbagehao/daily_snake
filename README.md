# Snake Odyssey - 蛇界空间 🐍

一个致力于探索和开发创意贪吃蛇网页版游戏的综合平台。

## 🌟 平台特性

*   **赛博朋克深色美学**：使用霓虹发光线条、毛玻璃设计和炫酷动画打造极致的视觉体验。
*   **实时合成声音引擎**：完全基于 Web Audio API 动态计算和合成音频信号，实现零延迟、高响应性的声效交互（无需下载外部音频）。
*   **60FPS 运动学粒子系统**：在 60 帧游戏循环中独立渲染平滑的拖尾粒子与碰撞爆炸特效。
*   **极简跨端操作**：完美适配 PC 键盘控制及移动端双端操作（触屏虚拟按键 + Canvas 滑动手势侦测）。
*   **完全独立的模块化设计**：每款贪吃蛇变体都是独立的静态站点，托管于 `games/` 子文件夹，能够实现不同画风与控制维度的完全自定义开发。

## 🎮 当前收录游戏

1.  **霓虹极战贪吃蛇 (`games/neon-snake/`)**：现代科技风格，支持超频药剂、黄金结晶等各类时间限制道具及碰撞粒子特效。
2.  **果园大作战 (`games/orchard-battle/`)**：经典复古的迷你手掌掌上游戏机风格，拟真纸质背景与多档位速度切换。

## 🚀 部署与运行

本地开发启动：
```bash
npm install
npm run dev
```

项目线上托管于 **Cloudflare Pages**。
